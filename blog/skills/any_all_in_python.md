**any** keyword evaluates to True if any one item in the supplied list is True or 1. For Example
```python
x = True
y = False
z = False

k = any([x,y,z])
print(k) # True

k = any([False, False, False]) # False
print(k)

k = any([0,0,0]) # False
print(k)

k = any([0<5, 20>50, 40*2==80]) # True
print(k)
```
